Recent Work

Harry Deaton Interiors Website

Ms. Deaton was interested in creating an online presence for her company, so we decided the single-page format would work great. The site is built on Bootstrap and fully responsive.

Ukule.li - a simple web app

One day my dad picked up a ukulele from a garage sale for me. I quickly figured out that there was no way I would ever remember the correct pitches for the strings. Thus, this tuner was born. It was a great opportunity to test out Dreamweaver's built-in tools for making responsive web pages. The tuner itself is built using Javascript's Audio object.

Responsive Menu Design

We were given the task of creating a one-page responsive design for class, and I thought it would be fun to make up a menu for a fictional cafe. It presented the challenge of formatting menu items that would resize elegantly along with the browser size. I was particularly happy with how the site turned out on the iPad mini. This was done from scratch using standards-based HTML5 and CSS3, and included a simple logo design.

Pearls of Europe Breeder

A longstanding local breeder needed a website to continue growing business. Vital to the owner was the ability to hop in and update the website, whether it be when a new litter was delivered or the final pup found a home. To meet that need, this design is built on Squarespace, with a few custom CSS and Javascript tweaks.
